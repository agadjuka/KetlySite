# Сравнение компонентов: Мобильная vs Десктопная версия

## 1. Поле ввода (ChatInput)

**Файл:** `src/components/chat/ChatInput.tsx`  
**Слой:** Компонент (UI)  
**Использование:**
- Десктоп: `src/components/layout/DesktopLayout.tsx` (строка 56)
- Мобильная: `src/components/layout/MobileLayout.tsx` (строка 44)

**Различия:**
- **Мобильная:** Добавлена кнопка со звездочкой (✨) слева от поля ввода для открытия меню быстрых действий (`onToggleMenu`). Кнопка скрыта на десктопе через `lg:hidden`.
- **Десктоп:** Кнопка звездочки отсутствует, поле ввода без дополнительных элементов.
- **Общее:** Один компонент, адаптация через условный рендеринг (`onToggleMenu && ...`). Enter отправляет сообщение только на десктопе (проверка `window.innerWidth > 768`).

## 2. Кнопка "Примеры"

**Десктоп:** `src/components/widgets/ExamplesButton.tsx`  
**Мобильная:** `src/components/widgets/MobileExamplesButton.tsx`  
**Слой:** Виджет (UI компонент)  
**Использование:**
- Десктоп: `src/components/layout/DesktopLayout.tsx` (строка 37) - в левой колонке, внизу
- Мобильная: `src/components/layout/MobileLayout.tsx` (строка 60) - фиксированная позиция справа внизу

**Различия:**

| Параметр | Десктоп | Мобильная |
|----------|---------|-----------|
| **Позиция** | Левая колонка, внизу (в flex-контейнере с `mt-auto`) | Фиксированная справа (`fixed right-4 bottom-46`) |
| **Внешний вид** | Полноразмерная кнопка с текстом "Примеры" | Круглая кнопка с иконкой звездочки (⭐) |
| **Размер** | `h-12`, `w-full` | `w-12 h-12` (круглая) |
| **Стили** | Класс `button-examples` (CSS в `globals.css`) | Класс `button-examples-mobile` |
| **Анимация** | Выпадающее меню вверх-вправо | Выпадающее меню вверх-влево |
| **Состояние** | Управляется из `DesktopLayout` (проп `isOpen`) | Внутреннее состояние (`useState`) |

## 3. Кнопка "Сообщения" / "Связаться с нами"

**Десктоп:** `src/components/widgets/ContactButton.tsx`  
**Мобильная:** `src/components/widgets/MobileContactButton.tsx`  
**Слой:** Виджет (UI компонент)  
**Использование:**
- Десктоп: `src/components/layout/DesktopLayout.tsx` (строка 40) - в левой колонке, внизу рядом с кнопкой "Примеры"
- Мобильная: `src/components/layout/MobileLayout.tsx` (строка 57) - фиксированная позиция справа внизу

**Различия:**

| Параметр | Десктоп | Мобильная |
|----------|---------|-----------|
| **Позиция** | Левая колонка, внизу (в flex-контейнере) | Фиксированная справа (`fixed right-4 bottom-32`) |
| **Текст** | "Связаться с нами" (`t.chat.contactUs`) | Иконка сообщения (чаты) |
| **Внешний вид** | Полноразмерная кнопка с текстом | Круглая кнопка с иконкой |
| **Размер** | `h-12`, `w-full` | `w-12 h-12` (круглая) |
| **Поведение** | При клике заменяется на иконки Telegram/WhatsApp внутри кнопки | При клике раскрываются иконки Telegram/WhatsApp рядом с кнопкой (анимация) |
| **Анимация** | Fade-in/zoom-in иконок внутри | Scale и slide иконок справа налево |

## Архитектурные слои

1. **Layout слой:** `src/components/layout/` - определяет структуру и позиционирование
2. **Widget слой:** `src/components/widgets/` - переиспользуемые UI компоненты
3. **Chat слой:** `src/components/chat/` - компоненты чата (включая ChatInput)
